# 画像生成ツール

カスタムサイズで画像を生成し、ダウンロードできるWebベースのツール。幅と高さを自由に指定して、様々な背景パターンの画像を作成できます。

## オンラインデモ

🌐 **https://so134.github.io/image-generator/** でライブデモをご利用いただけます。

## 機能

### サイズ設定
- 幅・高さを自由に入力可能（1〜4000px）
- デフォルト: 800×600px
- リアルタイムでサイズを調整

### 背景オプション
1. **グラデーション**: 6種類のカラーパターンからランダム生成
2. **単色**: カラーピッカーで自由に色を選択
3. **パターン**: 5種類のデザインパターン
   - ドット
   - ストライプ
   - グリッド
   - 波形
   - 円

### その他の機能
- Canvas API による高速な画像生成
- PNG形式でダウンロード
- 生成画像にサイズ表示
- レスポンシブデザイン対応
- ローカルのみで動作（外部API不要）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで http://localhost:4321 にアクセスして使用。

## 使い方

1. 幅と高さを入力欄に指定
2. 背景タイプを選択（グラデーション/単色/パターン）
3. 「画像を生成」ボタンをクリック
4. プレビューを確認
5. 「ダウンロード」ボタンで画像を保存

ファイル名は `image_{幅}x{高さ}_{タイムスタンプ}.png` 形式で自動生成されます。

## 技術仕様

- **フレームワーク**: Astro 4.x
- **画像生成**: Canvas API
- **スタイリング**: インラインCSS
- **JavaScript**: Vanilla JS（クライアントサイド）
- **外部依存**: なし（完全オフライン対応）
